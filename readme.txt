Инструкции:
  1. Создание нового блока:
    bem create blocks/pages/block-name

Правила:

  1. Иерархия блоков:
    Layout > Pages > Sections > Items

  2. Отступы не задаются независимым блокам.
      Как можно:
      - Наследование через элемент.
      - Добавление класса (можно сделать блок элементом).
      - Обертка (как блок, у которого роль, только в позиционировании) Обертки по иерархии (п.1).

  3. У последнего элемента группы, отступ обнуляется (всегда).

  4. Если можно использовать переменную - использовать переменную.
      - Все отступы между секциями задаются в файле variables.scss.
      - Все размеры шрифтов, заголовков, в том числе их значания в media.
      - Все цвета.
